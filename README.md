# Weather Daily Report

This project fetches weather data, generates a report, and outputs it as an HTML file.

## Project Structure

- `data/nws_stations.json`: List of weather stations.
- `output/index.html`: The generated HTML report.
- `src/`: Source code for fetching data and generating the report.
- `style/`: CSS styles for the report.
- `template/`: HTML template for the report.
- `run.py`: Main script to orchestrate the report generation.
